# Login & Logout System
Login and Logout System Using Session & Cookies with Golang Gin Framework

## Usage

### Start using it

```bash
go mod init <project_name>
```

Download and install dependencies:

```bash
go get -u github.com/gin-gonic/gin
```

```bash
go get github.com/gin-contrib/sessions
```

### Run Project

```bash
go run main.go
```
